# Pass Multi-Line text from SP to Report:

```sql
CREATE PROCEDURE dbo.MyReportProc
AS
BEGIN
    DECLARE @msg NVARCHAR(MAX)

    SET @msg =
        'First line' + CHAR(13) + CHAR(10) +
        'Second line' + CHAR(13) + CHAR(10) +
        'Third line'

    SELECT
        @msg AS ReportMessage
END
```

Then in SSRS:

Add a TextBox to the report
Set expression:

```vb
=Fields!ReportMessage.Value
```

Important TextBox settings:

CanGrow = True
Use a sufficiently large height/width
If needed, set Markup type = None

Usually CHAR(13) + CHAR(10) (CRLF) works best in SSRS 2008.

# Conditional display:

```vb
=IIF(
    IsNothing(Fields!SomeField.Value)
    OR Len(Trim("" & First(Fields!SomeField.Value, "Header"))) = 0,
    Nothing,
    vbCrLf & "Your application is being processed and "
    & CStr(Fields!SomeField.Value)
    & "." & vbCrLf
)
```
